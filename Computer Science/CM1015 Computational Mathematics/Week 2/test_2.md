$110 * 101$ in binary is **$11110$**.

---

How it's calculated:

The calculation was performed by converting the binary numbers to their decimal (base-10) equivalents, multiplying them, and then converting the result back to binary.

1.  **Binary to Decimal:**

    - $110_2 = (1 * 2^2) + (1 * 2^1) + (0 * 2^0) = 4 + 2 + 0 = 6_{10}$
    - $101_2 = (1 * 2^2) + (0 * 2^1) + (1 * 2^0) = 4 + 0 + 1 = 5_{10}$

2.  **Decimal Multiplication:**

    - $6 * 5 = 30_{10}$

3.  **Decimal to Binary:**

    - $30_{10} = 16 + 8 + 4 + 2 + 0 = (1 * 2^4) + (1 * 2^3) + (1 * 2^2) + (1 * 2^1) + (0 * 2^0) = 11110_2$
