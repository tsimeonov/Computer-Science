Creating an Express Server

1. Create directory

```
mkdir "3.1 Express Server"
```

2. Create index.js file

```
touch index.js
```

3. Initialise NPM

```
npm init -y
```

4. Install the Express packgage

```
npm i express
```

5. Write Server application in index.js

6. Start server
