## Sets

A `Set` is a built-in object that lets you store unique values of any type - primitives or object references.

1. Basic Usage

```js
const mySet = new Set();

mySet.add(1);
mySet.add(2);
mySet.add(2); // Duplicate, will be ignored
mySet.add("Hello");

console.log(mySet);
```

2. Useful Methods

| Method         | Description                                   |
| -------------- | --------------------------------------------- |
| `.add(value) ` | Adds a value (if it's not already in the set) |
|                |                                               |
